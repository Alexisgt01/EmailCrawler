# Email Crawler

Retrieves all email addresses from a google search. On all result pages (1 to xx pages)

## Installation

``git clone https://github.com/Alexisgt01/EmailCrawler crawler``

``cd crawler && composer install``

## Utilization

``php search "site internet"``
